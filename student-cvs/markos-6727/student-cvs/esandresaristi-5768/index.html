Comandos y Alias terminal

Comandos Utiles de la Terminal

  1. pwd: Muestra la ruta en la que nos encontramos
  2. mkdir: Crea una nueva carpeta con el nombre que le asignemos
  3. ls: Muestra los archivos de una ruta
  4. touch: Crea un nuevo archivo con la extensión que le demos
  5. cd: Nos permite navegar entre carpetas

Alias utiles

  1. alias instalar='sudo apt-get install': Nos permite la instalación de un paquete
  2. alias gh="history | grep": Encontrar un comando en el historial de la terminal mediante el comando gh
